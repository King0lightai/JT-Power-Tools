HOW WE GOT HERE

JULY 2024
Launch
v1.0 — THE BEGINNING
  • Initial public release on Chrome Web Store
  • 4 core features: Drag & Drop, Contrast Fix, Text Formatter, Dark Mode
  • Modular architecture designed for growth
AUGUST 2024
Major
v3.0 — CUSTOM THEMES
  • Custom Theme engine with RGB color sliders
  • Integrated contrast fix into custom themes
  • Collapsible sections in popup UI
SEPTEMBER 2024
Feature
v3.1 — SMART JOB SWITCHER
  • Keyboard-driven job navigation (J+S / Alt+J)
  • Full drag & drop refactor — 90% code reduction
  • Inline color previews for theme customization
OCTOBER 2024
Feature
v3.2 — BUDGET HIERARCHY
  • Budget hierarchy shading — 5 levels of depth visualization
  • HSL color space for adaptive theming
  • Dark mode + custom theme aware shading
NOVEMBER 2024
Major
v3.3 — QUICK NOTES & PREVIEW
  • Quick Notes — persistent notepad on every page
  • Preview Mode — live markdown rendering
  • Table builder for text formatter
  • Feedback link and improved popup UX
DEC 2024 — JAN 2025
Feature
v3.3.1–3.3.4 — CHECKBOXES & FREEZE
  • Action Items quick completion checkboxes
  • Month schedule task completion
  • Freeze Header — sticky job info while scrolling
  • Character counter for message fields
  • Most stable architecture release to date
DEC 2025 — JAN 2026
Beta
v3.3.6–3.3.10 — POLISH ERA
  • Auto Collapse completed schedule groups
  • Custom Theme overhaul with HSL palette generation
  • ToDo Drag & Drop for calendar due dates
  • Dozens of bug fixes across all features
JANUARY 2026
Major
v3.5 — TIERS & TEMPLATES
  • 4-tier licensing: Free, Essential, Pro, Power User
  • Message templates system with sync
  • Custom Field Filter for Power Users
  • AI / MCP integration panel
  • Kanban task checkboxes
FEBRUARY 2026
Major
v3.6 — RAPID FIRE
  • Microsoft Edge extension + mobile support
  • Availability Filter for schedule views
  • Takeoff Print Tool for PDF markup
  • Theme presets for one-click styling
  • Reverse Thread Order
  • Job context indicator & saved filters
  • Security hardening (SSRF, XSS, injection fixes)
MARCH 2026
Now
v3.6.7 — TODAY
  • Company Shared Templates (Essential+)
  • Preview Pinned Mode
  • MCP Server Beta for AI tool integration
  • Auto device registration
  • 27+ features across 4 tiers
