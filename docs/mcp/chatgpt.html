Connect ChatGPT to your JobTread data via MCP

ChatGPT MCP Support

ChatGPT's MCP support is currently in beta and available to ChatGPT Plus, Pro, and Team subscribers. Features may vary as OpenAI continues to develop this functionality.

What You'll Need

  • JT Power Tools Premium - Active subscription with MCP access
  • License Key - Your JT Power Tools license key
  • Grant Key - Your JobTread API grant key (from the API tab in the extension popup)
  • ChatGPT Plus/Pro/Team - MCP requires a paid ChatGPT subscription

Step 1: Get Your Credentials

  1. Open JT Power Tools Popup
    Click the extension icon in your Chrome toolbar while on JobTread
  2. Go to the API Tab
    Click the "API" tab to find your Grant Key
  3. Go to the License Tab
    Click the "License" tab to find your License Key
  4. Copy Both Keys
    You'll need both for the configuration

Step 2: Access ChatGPT Settings

  1. Open ChatGPT
    Go to chat.openai.com
  2. Open Settings
    Click your profile icon, then "Settings"
  3. Navigate to Features
    Look for "Features" or "Beta features" section
  4. Find MCP Servers
    Look for "MCP Servers" or "External Tools"

Step 3: Add JobTread MCP Server

In the MCP configuration, add a new server with these details:

Name JobTread
Endpoint URL https://mcp.jobtread-tools.pro/sse
Authentication Bearer Token
Token YOUR_LICENSE_KEY:YOUR_GRANT_KEY
Replace Placeholders

Make sure to replace YOUR_LICENSE_KEY and YOUR_GRANT_KEY with your actual keys!

Step 4: Enable the Connection

  1. Save Configuration
    Save your MCP server settings
  2. Enable JobTread
    Toggle the JobTread MCP server ON
  3. Start a New Chat
    Open a new chat to use the connected tools

Step 5: Test the Connection

In a new ChatGPT conversation, try these prompts:

  • "List my recent JobTread jobs"
  • "Show me open tasks for this week"
  • "What customers do I have in JobTread?"
  • "Summarize the budget for [job name]"
Mention JobTread

When asking questions, mention "JobTread" to help ChatGPT know to use the MCP tools. For example: "Using JobTread, show me my open jobs"

Available Tools

The MCP server provides these read-only tools:

Tool Description
search_jobs Search and list JobTread jobs
list_customers Get customer information
list_vendors Get vendor information
list_tasks Get tasks and to-dos
get_budgets Get budget details for jobs
knowledge_lookup Search knowledge base

Troubleshooting

MCP Option Not Available

  • Ensure you have ChatGPT Plus, Pro, or Team subscription
  • MCP support is being rolled out gradually - it may not be available in all regions yet
  • Try refreshing ChatGPT or clearing your browser cache

Connection Failed

  • Verify your License Key and Grant Key are correct
  • Check that your premium subscription is active
  • Ensure the URL is exactly https://mcp.jobtread-tools.pro/sse
  • Try disconnecting and reconnecting the MCP server

Tools Not Being Used

  • Explicitly mention "JobTread" in your prompts
  • Start a fresh conversation
  • Check that the MCP server is enabled in your settings
