Overview

The Schedule Task Checkboxes feature adds convenient checkboxes to your schedule task cards, allowing you to quickly mark tasks as complete directly from the month view. No more clicking into each task to update its status - just check the box and move on!

Key Features

  • One-Click Completion - Mark tasks complete instantly with a single click
  • Visual Feedback - Completed tasks show a checkmark and visual strike-through
  • Works in Month View - Checkboxes appear on task cards in the schedule month view
  • Syncs with JobTread - Status updates are saved to JobTread automatically
  • Undo Support - Uncheck to mark a task as incomplete again

Getting Started

Prerequisites

  • JT Power Tools extension installed in Chrome
  • Active Pro license
  • Schedule Task Checkboxes feature enabled in extension popup
  • Access to JobTread schedule view

Enabling the Feature

  1. Click the JT Power Tools extension icon in your Chrome toolbar
  2. Locate the "Schedule Task Checkboxes" toggle (marked with Pro badge)
  3. Toggle it ON - the switch will turn blue
  4. Navigate to your JobTread schedule page
  5. The checkboxes will now appear on your task cards

How to Use

Marking Tasks Complete

  1. Navigate to Schedule Month View
    Go to your schedule and ensure you're in the month view
  2. Find the Task Card
    Locate the task you want to mark as complete
  3. Click the Checkbox
    Click the checkbox on the task card - it will show a checkmark when complete
  4. Task Updates Automatically
    The task status syncs with JobTread in the background
Pro Tip

Use this feature at the end of each day to quickly check off all completed tasks from your schedule view without opening each one individually.

Unmarking Tasks

If you accidentally mark a task as complete or need to reopen it:

  1. Find the completed task (shows checkmark)
  2. Click the checkbox again to uncheck it
  3. The task will be marked as incomplete and synced with JobTread

Visual Indicators

Checkbox States

  • Empty Checkbox - Task is incomplete/in progress
  • Checked Checkbox - Task has been marked as complete

Task Card Appearance

When a task is marked complete:

  • The checkbox displays a checkmark
  • Visual styling indicates the completed status
  • The task remains visible on the schedule for reference

Best Practices

Efficient Workflow Tips

  1. End-of-Day Review - Check off completed tasks at the end of each workday
  2. Morning Planning - Review checked tasks from previous days to track progress
  3. Team Updates - Use checkboxes during team meetings to update task statuses in real-time

Common Use Cases

Daily Task Management

Quickly mark off tasks as your team completes them throughout the day without interrupting your workflow.

Weekly Reviews

During weekly planning sessions, review the month view and check off any tasks that were completed but not yet marked.

Project Tracking

Get a visual overview of project progress by seeing which scheduled tasks have been completed at a glance.

Troubleshooting

Checkboxes Not Appearing

Solution:

  • Verify the feature is enabled in the extension popup
  • Make sure you're in the schedule month view
  • Refresh the JobTread page
  • Check that your Pro license is active

Status Not Saving

Solution:

  • Check your internet connection
  • Refresh the page and try again
  • Open browser console (F12) to check for any error messages
  • Verify you have permission to edit the task in JobTread

Checkbox Unresponsive

Solution:

  • Make sure you're clicking directly on the checkbox, not the task card
  • Wait a moment for any pending operations to complete
  • Try refreshing the page
