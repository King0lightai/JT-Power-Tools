☑ Schedule Checkboxes

Premium Feature - Requires License Activation

Premium Feature

This feature requires a premium license. Learn how to activate your license.

Overview

The Schedule Checkboxes feature adds convenient checkboxes to your JobTread schedule, allowing you to quickly mark tasks and action items as complete directly from the schedule view. This streamlined workflow eliminates the need to open individual items just to check them off.

Key Features

  • Task Checkboxes - Mark tasks as complete directly from the schedule view
  • Action Item Checkboxes - Check off action items without opening them
  • Inline Updates - Changes save automatically in the background
  • Visual Feedback - Clear checkbox indicators show completion status
  • Time Saver - Complete items with a single click instead of multiple steps

Getting Started

Prerequisites

  • JT Power Tools extension installed in Chrome
  • Active premium license
  • Schedule Checkboxes feature enabled in extension popup
  • Access to JobTread schedule view

Enabling the Feature

  1. Click the JT Power Tools extension icon in your Chrome toolbar
  2. Locate the "Schedule Checkboxes" toggle (marked with Premium badge)
  3. Toggle it ON - the switch will turn blue
  4. Navigate to your JobTread schedule page
  5. The feature is now active and ready to use

How to Use

Task Checkboxes

Task checkboxes appear next to tasks in your schedule view:

  1. Navigate to Schedule
    Go to any job's schedule view in JobTread
  2. Locate the Checkbox
    Each task will have a checkbox displayed next to it
  3. Click to Complete
    Click the checkbox to mark the task as complete
  4. Automatic Save
    The status updates automatically - no need to save manually
Pro Tip

Use task checkboxes during morning standups or end-of-day reviews to quickly update multiple task statuses without navigating away from the schedule view.

Action Item Checkboxes

Action item checkboxes work similarly for action items:

  1. Find Action Items
    Action items in the schedule will display checkboxes
  2. Check Off Completed Items
    Click the checkbox when an action item is done
  3. See Updates Reflect
    The action item status updates in the background

Workflow Benefits

Faster Daily Updates

Instead of clicking into each task to mark it complete:

  • Before: Click task → Wait for dialog → Find completion option → Save → Close → Repeat
  • After: Click checkbox → Done!

Better Progress Visibility

  • See completion status at a glance on the schedule
  • Quickly identify what's done vs. what still needs attention
  • Update progress during team meetings in real-time

Common Use Cases

Morning Review

Start your day by reviewing the schedule and checking off any tasks that were completed the previous day or overnight.

Field Updates

Field workers can quickly mark tasks complete from their mobile browser without navigating through multiple screens.

End-of-Day Closeout

Project managers can efficiently close out the day by checking off all completed items in one view.

Team Standups

During daily standups, team leads can update task statuses in real-time as team members report their progress.

Visual Indicators

Checkbox States

  • Unchecked - Empty checkbox indicates task/action item is not complete
  • Checked - Filled checkbox indicates the item is marked complete

Loading State

When you click a checkbox:

  • The checkbox may briefly show a loading indicator
  • Once saved, the checkbox reflects the new state
  • If there's an error, the checkbox reverts to its previous state

Best Practices

Efficient Workflow Tips

  1. Batch Updates - Review the schedule and check off multiple items at once
  2. Regular Reviews - Make checkbox updates part of your daily routine
  3. Communicate Changes - Let team members know when you update their tasks

Troubleshooting

Checkboxes Not Appearing

Solution:

  • Verify the feature is enabled in the extension popup
  • Refresh the JobTread page
  • Check that your premium license is active
  • Make sure you're on a schedule page (not a different view)

Checkbox Click Not Saving

Solution:

  • Check your internet connection - updates require connectivity
  • Wait a moment and try again (may be temporary network issue)
  • Open browser console (F12) to check for any error messages
  • Try refreshing the page and attempting again

Status Shows Incorrectly

Solution:

  • Refresh the page to sync with the latest data
  • Check if another user updated the same item
  • Verify the change saved by opening the item details

Technical Details

How It Works

The extension works by:

  1. Detecting schedule items in the JobTread DOM
  2. Adding checkbox elements next to tasks and action items
  3. Listening for checkbox click events
  4. Sending status updates through JobTread's API
  5. Updating the visual state based on the API response

Compatibility

  • Works with JobTread's schedule views
  • Compatible with both light and dark mode
  • Supports tasks and action items

Limitations

  • Requires active internet connection for status updates
  • Only works on schedule page views
  • Cannot undo a completion directly (use task details to change status back)
