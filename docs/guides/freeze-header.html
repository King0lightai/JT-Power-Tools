Essential Feature - Keep Important Info Visible

Essential Feature

This feature requires an Essential license or higher. Learn how to activate your license.

Overview

Freeze Header keeps job information, navigation tabs, and action toolbars visible while scrolling through long pages. No more scrolling back to the top to see job details or access navigation tabs.

Key Features

  • Sticky Job Header - Job name and info stay at the top while scrolling
  • Sticky Navigation Tabs - Budget, Schedule, Messages tabs always accessible
  • Sticky Action Toolbars - Add, filter, and action buttons remain visible
  • Smart Layering - Elements stack correctly without overlapping
  • Multi-Page Support - Works on all major JobTread pages
  • Automatic Detection - Adapts to different page layouts automatically

How It Works

When you enable Freeze Header, the extension automatically detects which page you're on and makes the appropriate elements sticky. As you scroll down, these elements "freeze" at the top of your screen.

Supported Pages

  • Job Dashboard - Header and tabs stay visible
  • Budget View - Header, tabs, and toolbar freeze
  • Schedule View - Header, tabs, and Gantt headers freeze
  • Messages - Header and tabs remain accessible
  • Documents - Header and toolbar stay pinned
  • Daily Logs - Header and action bar freeze
  • Files Page - Works on both top-level and job-level files
  • Kanban Views - Column headers stay visible while scrolling

Getting Started

Enable Freeze Header

  1. Open JT Power Tools
    Click the extension icon in your Chrome toolbar while on JobTread
  2. Find Freeze Header
    Look for "Freeze Header" in the features list
  3. Toggle On
    Click the toggle to enable the feature
  4. Start Scrolling
    Navigate to any job page and scroll down to see headers freeze

Visual Behavior

What Stays Visible

When scrolling on a job page, these elements freeze in order from top to bottom:

  1. Top Header - JobTread main navigation bar
  2. Job Header - Job name, number, and status
  3. Tab Navigation - Budget, Schedule, Messages, etc.
  4. Action Toolbar - Add buttons, filters, view toggles

Visual Indicators

  • Frozen elements get a subtle shadow to show they're elevated
  • Background becomes solid white (or dark in Dark Mode) to prevent content showing through
  • Proper z-index ensures dropdowns and modals appear above frozen elements

Tips & Best Practices

Works Great With

  • Dark Mode - Frozen headers adapt to dark theme automatically
  • Custom Theme - Headers respect your custom colors
  • Budget Hierarchy Shading - Toolbar stays visible while scanning budget
  • Schedule Drag & Drop - Headers stay put while dragging items

Performance Notes

  • Uses CSS position: sticky for smooth, native performance
  • Automatically adjusts when window is resized
  • Re-applies when navigating between pages (SPA detection)
  • Minimal impact on page load time

Troubleshooting

Headers Not Freezing?

  • Make sure the feature is enabled in the extension popup
  • Try refreshing the page after enabling
  • Check if you're on a supported page type

Elements Overlapping?

  • This is rare but can happen with certain JobTread updates
  • Try toggling the feature off and on
  • Report issues on GitHub if the problem persists
