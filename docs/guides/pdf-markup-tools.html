Essential Feature - Enhanced PDF Annotation

Essential Feature

This feature requires an Essential license or higher. Learn how to activate your license.

Overview

PDF Markup Tools enhances JobTread's built-in PDF annotation system with professional stamps and an improved eraser tool. Perfect for construction plan markup, document review, and redlining drawings.

Key Features

  • Professional Stamp Library - Pre-built stamps for construction, electrical, plumbing, and HVAC markup
  • Approval Stamps - APPROVED, REVIEWED, REJECTED, PENDING, CONFIDENTIAL
  • Date Stamps - Automatically include today's date
  • Architecture Symbols - Doors, windows, electrical outlets, plumbing fixtures, and more
  • Enhanced Eraser - Improved eraser tool for cleaning up annotations
  • Dark Mode Compatible - Works with JT Power Tools dark mode

Getting Started

Prerequisites

  • JT Power Tools extension installed
  • Active Essential license or higher
  • Access to JobTread PDF viewer

Enabling PDF Markup Tools

  1. Click the JT Power Tools extension icon
  2. Locate "PDF Markup Tools" (marked with Essential badge)
  3. Toggle it ON
  4. Open any PDF in JobTread's document viewer
  5. The enhanced toolbar appears with new stamp and eraser options

Using the Tools

Placing a Stamp

  1. Open a PDF in JobTread's document viewer
  2. Click the Stamps button in the toolbar
  3. Select a stamp category (Approval, Date, Architecture)
  4. Click on the stamp you want to use
  5. Click on the PDF where you want to place it

Using the Eraser

  1. Click the Eraser tool in the toolbar
  2. Click on annotations you want to remove
  3. The enhanced eraser provides better selection accuracy
Pro Tip

Use the VERIFY stamp to highlight items that need field confirmation, and the NOTE stamp for general comments. This creates a clear markup workflow for your team.

Integration with JobTread

PDF Markup Tools integrates seamlessly with JobTread's native annotation system:

  • Persistence - All stamps are saved with JobTread's annotation system
  • Undo/Redo - Full undo/redo support through JobTread
  • Sharing - Annotations are visible to all team members with access
  • Printing - Stamps appear when printing annotated PDFs

Common Use Cases

Plan Review

Mark up construction drawings during plan review:

  • Use electrical stamps to mark outlet and switch locations
  • Add plumbing symbols for fixture placement
  • Use VERIFY stamps for items needing field confirmation
  • Add NOTE stamps for special instructions

Document Approval

Streamline document approval workflows:

  • Use APPROVED/REJECTED stamps for quick decisions
  • Add date stamps to track when documents were reviewed
  • Use PENDING for items awaiting additional information

Clearly mark changes on drawings:

  • Use ADD stamps for new items
  • Use REMOVE stamps for items being deleted
  • Use RELOCATE stamps for items being moved
  • Use REVISION stamps to mark changed areas

Troubleshooting

Stamps Not Appearing

Solution:

  • Verify PDF Markup Tools is enabled in extension popup
  • Check that your Essential license is active
  • Refresh the PDF viewer page
  • Make sure you're viewing a PDF in JobTread (not an external viewer)

Toolbar Not Showing

Solution:

  • The toolbar appears when you open a PDF for annotation
  • Click "Edit" or the annotation mode button if available
  • Try refreshing the page
