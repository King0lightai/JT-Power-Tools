Free Feature - Track Characters & Save Time with Templates

Overview

Message Counter & Templates is a powerful productivity feature that combines real-time character counting with reusable message templates. Track your character count as you type messages, and save frequently used text as templates for quick insertion.

Key Features

  • Real-Time Character Count - See your character count update as you type in message fields
  • Message Templates - Create, save, and paste signatures and common messages
  • Color-Coded Warnings - Visual alerts as you approach character limits
  • One-Click Insert - Paste saved templates instantly into any message
  • Unlimited Templates - Save as many templates as you need
  • Synced Storage - Templates sync across your devices via Chrome

Message Templates

Templates let you save frequently used text snippets like signatures, common responses, and standard messages. Insert them with a single click instead of typing the same content repeatedly.

Creating a Template

  1. Open the Templates Panel
    Click the templates icon in the message toolbar (appears when you focus on a message field)
  2. Click "Add New Template"
    A dialog will appear for creating your template
  3. Enter a Name
    Give your template a descriptive name (e.g., "Email Signature", "Thank You Response")
  4. Enter the Template Content
    Type or paste the text you want to save
  5. Save the Template
    Click Save - your template is now ready to use

Using a Template

  1. Focus on a Message Field
    Click in any message or text field in JobTread
  2. Open the Templates Panel
    Click the templates icon in the toolbar
  3. Select Your Template
    Click on the template you want to insert
  4. Template is Inserted
    The template text is pasted at your cursor position
Pro Tip

Create templates for your email signature, common project updates, standard customer responses, and frequently used instructions. This can save hours of typing each week!

Managing Templates

  • Edit - Click the edit icon next to a template to modify it
  • Delete - Click the delete icon to remove a template
  • Reorder - Templates appear in the order they were created

Character Counter

The character counter displays a real-time count of characters in message fields, helping you stay within JobTread's limits.

Where It Appears

The character counter is visible in message fields throughout JobTread, including:

  • Customer messages
  • Internal messages
  • Comments

Visual Feedback

The counter changes color based on how close you are to the limit:

Status Color Meaning
Safe Gray Plenty of space remaining
Warning Orange Approaching the limit
Danger Red Very close to limit
Over Limit Dark Red Exceeded limit - trim your message

Display Format

  • Normal: 123 / 4096 - Current count / Maximum
  • At Limit: Limit reached
  • Over Limit: 50 over limit - Shows how many characters to remove

Getting Started

  1. Enable the Feature
    Open JT Power Tools popup and enable "Message Counter & Templates"
  2. Navigate to Messages
    Go to any page with a message field in JobTread
  3. Start Using
    The counter appears automatically, and the templates icon shows in the toolbar

Common Use Cases

Customer Communications

Use templates for standard customer messages like project updates, payment reminders, and thank you notes. The character counter ensures your messages won't get cut off.

Team Signatures

Create a professional signature template that includes your name, title, and contact information. Insert it at the end of every customer-facing message.

Standard Responses

Save time by creating templates for frequently asked questions or common scenarios. Customize as needed after inserting.

Tips & Best Practices

Template Tips

  • Use descriptive names so you can quickly find the right template
  • Include placeholders like [CLIENT NAME] for content you'll customize
  • Review and update templates periodically to keep them current
  • Create templates for both internal and external communications

Character Counter Tips

  • Watch for the orange warning - it's your cue to start wrapping up
  • If you go over, the "X over limit" message tells you exactly how much to cut
  • Consider splitting long messages into multiple messages

Troubleshooting

Templates Not Saving?

  • Make sure the feature is enabled in the extension popup
  • Check that you have storage permissions enabled for the extension
  • Try refreshing the page and creating the template again

Counter Not Appearing?

  • Make sure the feature is enabled in the extension popup
  • The counter only appears in message fields, not all text fields
  • Try clicking directly in the message field

Template Not Inserting?

  • Make sure your cursor is in the text field before selecting a template
  • Try clicking in the field first, then opening the templates panel
