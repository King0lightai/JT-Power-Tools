Essential Feature - Your Personal Notepad

Essential Feature

This feature requires an Essential license or higher. Learn how to activate your license.

Overview

Quick Notes provides a persistent notepad accessible from any JobTread page. Create, edit, search, and organize multiple notes with rich text formatting. Access it instantly with the Q + N keyboard shortcut or click the icon in the top header bar.

Key Features

  • Always Accessible - Icon in the top header bar is visible on every page
  • Quick Keyboard Access - Press Q+N to open from anywhere
  • Multiple Notes - Create and manage unlimited notes
  • Rich Text Formatting - Bold, italic, underline, strikethrough
  • Bullet Points & Checkboxes - Organize with lists and task checkboxes
  • Full Undo/Redo - Up to 50 state changes remembered
  • Search Notes - Find notes by title or content
  • Import/Export - Backup and restore notes as JSON files
  • Syncs Across Devices - Notes save to your Chrome account
  • Resizable Panel - Drag to adjust panel width

How to Access Quick Notes

Option 1: Keyboard Shortcut (Recommended)

The fastest way to open Quick Notes:

  1. Press Q then N
    Press Q followed by N in quick succession (within 500ms)
  2. Panel Opens
    The Quick Notes sidebar slides in from the right
  3. Press Again to Close
    Use Q+N again or press Esc to close
Pro Tip

The Q+N shortcut works from any page in JobTread - you don't need to navigate anywhere first. It's the quickest way to jot down a note while on a call or in a meeting.

Option 2: Header Icon (Always Visible)

Quick Notes adds an icon to JobTread's top header bar, right next to the other header icons like Time Clock and Notifications:

  1. Look for the Notes Icon
    Find the notepad/pencil icon in the top-right header area
  2. Click to Open
    Click the icon to open the Quick Notes panel
  3. Click Again to Close
    Click the icon again or press Escape to close
Always There

The header icon is always visible no matter where you are in JobTread - on job pages, the schedule, budgets, or anywhere else. You never have to scroll or navigate to find it.

Creating and Managing Notes

Creating Your First Note

  1. Open Quick Notes
    Press Q+N or click the header icon
  2. Click "New Note"
    Or use the + button at the top of the sidebar
  3. Enter a Title
    Give your note a descriptive name
  4. Start Writing
    Use the editor to add your content
  5. Auto-Save
    Notes save automatically as you type

Sidebar View

  • See all your notes in a list
  • Search bar to filter notes by title or content
  • Click any note to open it in the editor
  • Notes sorted by last modified date

Editor View

  • Rich text toolbar with formatting buttons
  • Large editing area for your content
  • Back button to return to sidebar
  • Delete button to remove current note

Resizing the Panel

  • Drag the left edge of the panel to resize
  • Min width: 320px
  • Max width: 1200px
  • Your preferred width is remembered between sessions

Formatting Options

Text Styles

  • Bold - Make text stand out
  • Italic - Emphasize text
  • Underline - Highlight important items
  • Strikethrough - Mark completed items

Lists

  • Bullet Points - Unordered lists for general items
  • Checkboxes - Task lists with clickable checkboxes
  • Nesting - Use Tab to indent items under others

Links

Insert clickable links using Ctrl/Cmd + K. Links open in a new tab when clicked.

Import & Export

Exporting Notes

  1. Click the Menu Icon
    Found in the top-right of the sidebar
  2. Select "Export Notes"
    Downloads a JSON file with all your notes
  3. Save the File
    Keep it as a backup or transfer to another computer

Importing Notes

  1. Click "Import Notes"
    From the menu dropdown
  2. Select Your File
    Choose a previously exported JSON file
  3. Choose Import Mode
    • Merge - Add imported notes to existing ones
    • Replace - Delete all existing notes and import

Tips & Best Practices

Organization Tips

  • Use descriptive titles for easy searching
  • Create separate notes for different jobs or topics
  • Use checkboxes for to-do items
  • Export regularly for backup

Workflow Ideas

  • Meeting Notes - Press Q+N during client calls to quickly jot down notes
  • Job Checklists - Pre-visit checklists for each job
  • Templates - Save common text you reuse
  • Quick Reference - Material specs, codes, contacts

Troubleshooting

Q+N Shortcut Not Working

  • Make sure Quick Notes is enabled in the extension popup
  • Check that your Essential license is active
  • Try pressing Q and N more quickly (within 500ms)
  • Make sure focus isn't in a text input field
  • Refresh the page and try again

Header Icon Not Showing

  • Verify the feature is enabled in the extension popup
  • Refresh the JobTread page
  • The icon appears next to other header icons (Time Clock, Notifications)

Notes Not Saving

  • Check your internet connection
  • Verify the extension has storage permissions
  • Try exporting your notes as a backup
