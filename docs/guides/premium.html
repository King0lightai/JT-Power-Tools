License Verification

How Verification Works

When you activate your license:

  1. Extension sends your license key to Gumroad's API
  2. Gumroad verifies the key is valid and active
  3. Response confirms license status
  4. Extension stores verification locally
  5. Premium features are unlocked

Troubleshooting License Issues

License Key Not Working

Solution:

  • Double-check you copied the entire key (no extra spaces)
  • Verify you purchased from the correct Gumroad page
  • Check if your payment was successful
  • Try copying and pasting the key again
  • Contact Gumroad support if payment issues

"Invalid License" Error

Solution:

  • Ensure you're entering the license key, not order number
  • Check for typos or missing characters
  • Verify the license hasn't been refunded or disputed
  • Try activating on a different browser to test

"Connection Error" During Activation

Solution:

  • Check your internet connection
  • Verify Gumroad.com is accessible (not blocked by firewall)
  • Try again in a few minutes (could be temporary Gumroad issue)
  • Check if corporate firewall is blocking api.gumroad.com

Premium Features Not Working After Activation

Solution:

  • Verify you see "License Active" in popup
  • Check that premium feature toggles are turned ON
  • Refresh the JobTread page
  • Try disabling and re-enabling the feature
  • Deactivate and reactivate your license

License Shows Active But Features Locked

Solution:

  • Close and reopen the extension popup
  • Refresh the JobTread page
  • Check browser console (F12) for errors
  • Try clicking the premium feature toggle off and on
  • Report the issue on GitHub with details
