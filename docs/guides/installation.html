Quick Start

Installing JT Power Tools is simple. Follow the steps below to get started.

Install from Chrome Web Store

Install from Chrome Web Store

  1. Click the link above
    Visit the JT Power Tools page on the Chrome Web Store
  2. Click "Add to Chrome"
    Click the blue "Add to Chrome" button
  3. Confirm the installation
    Accept the permissions when prompted
  4. Start using
    The extension icon will appear in your Chrome toolbar

First-Time Configuration

Testing the Extension

  1. Navigate to JobTread
    Open app.jobtread.com
  2. Open Extension Popup
    Click the JT Power Tools icon in your Chrome toolbar
  3. Verify Features
    You should see a popup with toggle switches for all 7 features

Enabling Features

  1. Choose Features
    Toggle ON the features you want to use
  2. Features Activate
    Most features activate instantly without page reload
  3. Refresh if Needed
    If a feature doesn't seem active, click "Refresh Current Tab" in the popup
Which Features Should I Enable?

Start with free features like Text Formatter, Smart Job Switcher, and Dark Mode. Once comfortable, explore premium features like Schedule Drag & Drop and Custom Theme.

System Requirements

Chromium Based Browser

  • Chrome
  • Edge
  • Brave
  • Opera
  • Shift

JobTread Account

  • Active JobTread account required
  • Works with all JobTread subscription tiers
  • No special permissions needed from JobTread admin

System

  • OS - Windows, macOS, Linux, Chrome OS
  • RAM - 4GB minimum (extension uses minimal resources)
  • Internet - Required for JobTread and premium license verification

Permissions Explained

JT Power Tools requests the following permissions:

Permission Purpose Required?
storage Save your feature preferences and settings locally Yes
activeTab Apply features to current JobTread tab only when active Yes
app.jobtread.com Enable features on JobTread application only Yes
api.gumroad.com Verify premium license keys (optional - only if using premium features) No
Privacy First

JT Power Tools does NOT collect, store, or transmit your JobTread data. All settings are stored locally in your browser. Read our Privacy Policy

Updating the Extension

When installed from the Chrome Web Store, updates are automatic. You'll receive the latest version within 24-48 hours of release.

Troubleshooting

Features Don't Work on JobTread

Solution:

  • Verify you're on app.jobtread.com (not other JobTread domains)
  • Check that features are toggled ON in popup
  • Refresh the JobTread page
  • Try removing and re-installing the extension
  • Restart Chrome

Uninstalling

How to Uninstall

  1. Open Extensions Page
    Navigate to chrome://extensions
  2. Find JT Power Tools
    Locate the extension in your list
  3. Click "Remove"
    Click the "Remove" button
  4. Confirm Removal
    Confirm you want to remove the extension

What Gets Deleted

  • All extension files and code
  • Your saved settings and preferences
  • Premium license key (you'll need to re-enter if you reinstall)
  • Saved custom themes (Premium)

What Stays

  • Your JobTread data (unaffected)
  • Any formatting you applied to text (stored in JobTread, not extension)
  • Your premium license itself (still valid, just re-enter key)
