Power User Feature - Filter Jobs by Custom Fields

Power User Feature

This feature requires a Power User license and API configuration. Learn how to activate your license.

Overview

Custom Field Filter enhances the Quick Job Switcher by allowing you to filter jobs based on your organization's custom fields. Search and filter jobs by project manager, job type, status, or any other custom field you've set up in JobTread.

Key Features

  • Custom Field Search - Filter jobs by any custom field values
  • Enhanced Job Switcher - Integrates with Quick Job Switcher sidebar
  • Real-Time Filtering - Results update as you type
  • Multiple Field Support - Search across multiple custom fields simultaneously

Requirements

Prerequisites

  • JT Power Tools extension installed
  • Active Power User license
  • Quick Job Switcher enabled (required)
  • JobTread API Grant Key configured (required)
API Configuration Required

Custom Field Filter requires access to the JobTread API to retrieve custom field data. You must configure your Grant Key in the extension settings before this feature will work.

Setting Up

Step 1: Configure API Access

  1. Open the JT Power Tools popup
  2. Go to the "License" tab
  3. Scroll to the API Configuration section
  4. Enter your JobTread Grant Key
  5. Click "Test Connection" to verify

Step 2: Enable the Features

  1. Enable "Quick Job Switcher" (Essential feature)
  2. Enable "Custom Field Filter" (Power User feature)
  3. Refresh JobTread to apply changes

Using Custom Field Filter

Filtering Jobs

  1. Press J + S to open the Job Switcher
  2. Use the filter options to select custom field criteria
  3. Type in the search to further narrow results
  4. Press Enter to select a job

Common Filter Examples

  • By Project Manager - Filter to see only your assigned jobs
  • By Job Type - Show only residential, commercial, or specific project types
  • By Status - Filter by custom status fields
  • By Region/Territory - Filter jobs by geographic area

Best Practices

Effective Filtering

  • Set up meaningful custom fields in JobTread first
  • Use consistent naming conventions for custom field values
  • Combine filters with text search for precise results

Performance Tips

  • The feature caches custom field data for faster searches
  • Initial load may take a moment to fetch field data
  • Filters update automatically when job data changes

Troubleshooting

Filters Not Appearing

Solution:

  • Verify your API Grant Key is configured and tested
  • Check that Quick Job Switcher is enabled
  • Ensure Custom Field Filter is enabled
  • Refresh the JobTread page

No Custom Fields Showing

Solution:

  • Verify your organization has custom fields set up in JobTread
  • Check that your API key has permission to read custom fields
  • Try the "Test Connection" button to verify API access

API Connection Failed

Solution:

  • Double-check your Grant Key is entered correctly
  • Ensure the Grant Key has not expired
  • Verify you have the correct permissions in JobTread
  • Contact your JobTread administrator if issues persist
