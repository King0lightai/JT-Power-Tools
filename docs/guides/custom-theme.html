Pro Feature - Personalize Your JobTread Experience

Pro Feature

This feature requires a Pro license. Learn how to activate your license.

Overview

Custom Theme lets you personalize JobTread with your own color palette. Create themes that match your brand, reduce eye strain, or simply reflect your personal style. Save up to 3 custom themes for quick switching.

Key Features

  • Full Color Control - Choose primary, background, and text colors
  • Inline Previews - See colors before applying
  • Intelligent Coordination - Colors coordinate across entire interface
  • Task Color Preservation - Maintains task type colors with enhanced visibility
  • 5px Colored Borders - Clear task identification
  • Save 3 Themes - Quick switching between favorite palettes
  • Yellow Highlight Preservation - Edited budget cells stay highlighted
  • Seamless Task Cards - Blend with custom background

Getting Started

Prerequisites

  • JT Power Tools extension installed
  • Active premium license
  • Custom Theme enabled in popup

Accessing Theme Customization

  1. Click the JT Power Tools extension icon
  2. Find "Custom Theme" (marked with Premium badge)
  3. Toggle it ON
  4. Click "Customize Theme" button to open color picker panel

Creating Your Theme

Color Options

Primary Color

Used for:

  • Buttons and call-to-action elements
  • Links and interactive elements
  • Accent colors throughout interface
  • Task card borders

Background Color

Used for:

  • Main page backgrounds
  • Card backgrounds
  • Panel backgrounds
  • Task card backgrounds

Text Color

Used for:

  • All text content
  • Labels and headings
  • Form inputs
  • Navigation

Choosing Colors

  1. Open Color Picker
    Click the color preview box next to each color option
  2. Select Your Color
    Use the color picker to choose your preferred color. You'll see an inline preview.
  3. Verify Contrast
    Ensure text is readable against your chosen background
  4. Apply Theme
    Click "Apply" to see your theme across JobTread
Contrast Tip

Ensure your text color has good contrast with your background color. Dark text on light backgrounds, or light text on dark backgrounds work best.

Saving & Managing Themes

Saving a Theme

  1. Create your custom color palette
  2. Click "Save Theme" button
  3. Give your theme a name (e.g., "Ocean Blue", "Forest Green")
  4. Theme is saved to one of your 3 available slots

Loading a Saved Theme

  1. Open the theme customization panel
  2. View your saved themes in the "Saved Themes" section
  3. Click on a saved theme to apply it instantly

Deleting a Theme

  1. Find the theme you want to delete in "Saved Themes"
  2. Click the "Delete" or "√ó" button next to it
  3. Confirm deletion
  4. Slot is now available for a new theme

Troubleshooting

Colors Not Applying

Solution:

  • Verify Custom Theme is enabled
  • Check that Contrast Fix and Dark Mode are disabled
  • Click "Apply" after choosing colors
  • Refresh the page

Text Hard to Read

Solution:

  • Adjust text color for better contrast with background
  • Use darker text on light backgrounds, lighter text on dark backgrounds
  • Try a different background color

Saved Themes Won't Load

Solution:

  • Check that you have premium license activated
  • Verify themes were saved successfully
  • Try re-saving the theme
  • Clear browser cache and reload extension
