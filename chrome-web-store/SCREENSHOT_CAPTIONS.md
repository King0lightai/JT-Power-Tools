## Tips for Taking Screenshots

**Resolution:**
- 1280x800 pixels (recommended)
- Or 640x400 pixels (minimum)

**Format:**
- PNG (preferred for crisp text)
- JPEG (acceptable)

**Quality:**
- High quality, no blurriness
- Clear, readable text
- Good contrast
- Professional appearance

**Content:**
- Real JobTread interface (use test data)
- No sensitive information
- No client names or real project data
- Clean, uncluttered views

**Annotations (Optional):**
- Add subtle arrows or highlights
- Circle key features
- Keep annotations minimal
- Use professional colors (avoid Comic Sans!)

---

## Screenshot Taking Process

1. **Setup:**
   - Open JobTread in Chrome
   - Install JT Power Tools
   - Navigate to schedule or budget page
   - Set browser window to 1280x800

2. **Capture:**
   - Use built-in screenshot tool (Cmd/Ctrl + Shift + 5 on Mac)
   - Or Chrome DevTools Device Mode
   - Or extension like Awesome Screenshot

3. **Edit:**
   - Crop to exact size
   - Blur any sensitive data
   - Add subtle highlights if needed
   - Save as PNG

4. **Review:**
   - Check text is readable
   - Verify no sensitive data visible
   - Ensure professional appearance
   - Test on different screen sizes
