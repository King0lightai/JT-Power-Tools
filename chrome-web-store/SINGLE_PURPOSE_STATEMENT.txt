SINGLE PURPOSE STATEMENT (for Chrome Web Store submission)

JT Power Tools enhances the JobTread construction management platform with productivity features including drag-and-drop schedule management, dark mode for reduced eye strain, rich text formatting for budget descriptions, improved text contrast for better readability, quick job switching via keyboard shortcuts, custom color themes for personalization, and budget hierarchy visualization for better organization. All features are designed to improve workflow efficiency for construction project managers using JobTread.
