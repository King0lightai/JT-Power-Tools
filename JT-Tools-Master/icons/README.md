# Icons

This directory should contain the extension icons:

- `icon16.png` - 16x16 pixels (browser toolbar)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create these icons using any image editor. Recommended designs:

- **Theme**: Tools/wrench icon or "JT" monogram
- **Colors**: Match the extension's purple gradient (#667eea to #764ba2)
- **Style**: Modern, clean, professional

## Temporary Solution

If you don't have icons yet, the extension will still work but may show a default placeholder icon. You can:

1. Use any PNG images as placeholders (just name them correctly)
2. Use an online icon generator like:
   - https://favicon.io/
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/

## Quick Icon Creation

For a quick solution, you can create simple colored squares:
1. Create 16x16, 48x48, and 128x128 pixel PNG files
2. Fill with your brand color
3. Add white text "JT" in the center
4. Save as icon16.png, icon48.png, icon128.png
